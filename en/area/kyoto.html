Yes. Set A (fridge, washer, microwave) is ¥45,100 for one year, Set B (+ single bed) ¥55,080, Set C (+ semi-double bed) ¥72,320 — all period totals, plus only the one-time Kyoto delivery fee.

Yes, it's free in Kyoto too. Just send one message to book a collection date.
